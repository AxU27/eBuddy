# eBuddy: Service book app
## About
eBuddy is a drivers companion app with a separate service book for each vehicle with the ability to attach image and pdf files
so the user won't have to keep their physical receipts safe or have e-receipts scattered around their email, phone and computer.
The project uses .NET Maui and SQLite.

## Currently planned features
### Range calculator
A range calculator for EVs which uses information like the vehicles battery capacity, SOH, SOC and power consumption to calculate
the remaining range of the vehicle. Useful for some EVs with inaccurate range calculations, like older Kia Soul EVs
(personal experience)

### EV charger map
Fast and responsive map view of all publicly available chargers with price information, that can be filtered by charger type.
Trying to solve the problem of having to jump from app to app to find the best chargers for your journey. A feature lacking
from older EVs.
